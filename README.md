# docker-php-8-0-lamp
Docker container with php-8-0, apache, phpmyadmin & mariadb 

### Usage

* How to run it
```
	docker-compose -f php8-docker-compose.yml up -d --remove-orphans
```
* How to stop it
```
	docker-compose -f php8-docker-compose.yml down
```
